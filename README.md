# REMPLACER — Agarose Mouse Brain Mould / Moule cerveau de souris en agarose

**FR** | [EN below](#english-version)

## Version française

Ce dépôt contient les fichiers STL du moule en deux parties permettant de fabriquer des cerveaux de souris en agarose, utilisés dans l'atelier de médiation scientifique **REMPLACER** depuis 2017.

Le moule a été conçu à partir de données anatomiques réelles de souris, librement accessibles en ligne. Il permet de produire un modèle translucide compatible avec un cadre stéréotaxique standard, sans utiliser aucun animal.

### Contenu du dépôt

```
/stl/
    mould_part1.stl       — première moitié du moule
    mould_part2.stl       — deuxième moitié du moule
PROTOCOL.md               — protocole de fabrication des cerveaux en agarose
LICENSE                   — licence CC BY-NC-SA 4.0
```

### Protocole rapide

1. Imprimer les deux parties du moule (PLA, résolution 0.2 mm recommandée)
2. Préparer une solution d'agarose à 3% dans de l'eau distillée
3. Fermer le moule en scotchant les deux parties sur la tranche, puis ajouter des bandes de scotch à 90° pour assurer l'étanchéité, avant de couler à chaud
4. Laisser polymériser environ 15 min à température ambiante, puis 20 min minimum à 4°C — à ajuster selon vos conditions
5. Démouler délicatement

Le protocole complet est disponible dans [`PROTOCOL.md`](PROTOCOL.md) et sur le site du projet : [https://g0ub.github.io/remplacer/](https://g0ub.github.io/remplacer/)

## English version

This repository contains the two-part STL files for casting agarose mouse brain models used in the **REMPLACER** science outreach workshop since 2017.

The mould was designed from real mouse anatomical data, freely available online. It produces a translucent model compatible with a standard stereotaxic frame — no animals required.

### Repository contents

```
/stl/
    mould_part1.stl       — first half of the mould
    mould_part2.stl       — second half of the mould
PROTOCOL.md               — agarose brain casting protocol
LICENSE                   — CC BY-NC-SA 4.0 licence
```

### Quick protocol

1. Print both mould halves (PLA, 0.2 mm resolution recommended)
2. Prepare a 3% agarose solution in distilled water
3. Pour while hot into the closed mould, held together with clips
4. Allow to set for 30 min at room temperature, then 1h in the fridge
5. Demould carefully

Full protocol in [`PROTOCOL.md`](PROTOCOL.md) and on the project website: [https://g0ub.github.io/remplacer/](https://g0ub.github.io/remplacer/)
